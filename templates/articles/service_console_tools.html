Quick text is a predefined message, like a greeting, note, phrase, or answer to a common question. Users can insert quick text in their emails, chats, and more with the click of a button or a keyboard shortcut.

Great For: Standardization and replacing repetitive copy/paste tasks.

Designing Your Quick Text Strategy

Before implementing, consider these questions:

  • Who should be able to use quick text in their communications with customers, and on which channels?
  • Who should be able to create quick text messages for themselves?
  • Who should be able to create quick text messages for multiple users?
  • Do you want to create unique sets of quick text messages for different business units?
  • In which scenarios does your team need quick text messages?

How to Enable

Go to Setup → Quick Text Settings.

Note on Permissions: A user’s profile typically has default read permission, meaning users can insert quick text without additional setup. To give users more permission (like the ability to create quick text), you update their user profile or use a permission set.

Real World Scenarios

Scenario 1: Closing Cases

Service reps need to send emails to their customers before closing the case.

Scenario 2: Manager Tasks
